# Atividade Prática 02 DCC012-2020.3-A
Repositório para atividade prática 2 da disciplina Estrutura de Dados 2.

## Instruções para compilação e excução

```
$ make
$ ./encode [-i ARQUIVO_DE_ENTRADA] [-o ARQUIVO_DE_SAIDA]
$ ./decode [-i ARQUIVO_DE_ENTRADA] [-o ARQUIVO_DE_SAIDA]
```

Por padrão ambos os programas leem da `stdin` e escrevem na `stdout`.
